# Notion 导入 & 使用说明（快速开始）

1. 下载并解压 `frontend_interview_pack.zip`。  
2. 在 Notion 中创建一个页面，选择 “Import” -> “Markdown & CSV”，批量导入本目录下的 .md 文件。  
3. 导入后你可以把各个题库转为数据库、或用「Toggle / Heading」整理成学习卡片。  
4. 推荐结构：
   - Dashboard（学习进度）  
   - 面试题库（Database）  
   - 项目（Repo + Demo + 文档）  
   - 周计划（Calendar）  
5. 我已在 README 中给出每个文件的用途，导入后直接开始使用。

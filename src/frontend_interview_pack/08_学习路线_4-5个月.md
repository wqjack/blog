# 4-5 个月学习路线（详细周计划）

## 总体节奏
- 周一至周五：每天 2 小时（通勤/晚间背题 & 核心阅读）  
- 周末两天：各 10 小时（深度实战、项目开发、系统设计）

## Week 1-4：基础强化
- Week1: JS 原型、this、闭包、事件循环、手写题
- Week2: Promise、async/await、深拷贝、浏览器回流/重绘
- Week3: CSS 高级：Flex/Grid、动画、移动端适配
- Week4: 小项目复现 + 输出复习笔记

## Week 5-12：框架与工程化
- React 深入（hooks、fiber、性能）
- Next.js SSR/SSG/RSC
- 工程化（Vite/Webpack、Monorepo、CI）
- 中期项目：AI 文档问答基础版

## Week 13-16：全栈与性能
- Node/Nest 简单后端（Prisma、Postgres）
- 性能调优、监控（LCP、TTI）
- 项目迭代：加入 embedding 与向量检索

## Week 17-20：AI 应用与冲刺
- LLM API（function call、streaming）
- Vercel AI SDK 与 RAG
- 上线项目并准备面试展示
- 模拟面试与行为题练习

## 输出目标
- 至少 1 个可演示的 AI 项目（带 README / Demo / 代码）  
- 完整 Notion 面试题库（覆盖 JS/浏览器/React/网络/系统设计/手写）  
- 20 周计划已打卡记录
